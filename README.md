1. Для запуска сервиса можно воспользоваться файлом-запускатором RunTranslator.bat, который находится в каталоге target

2. Чтобы получить перевод текста можно воспользоваться двумя способами:
	а) Создать запрос вида:
		localhost:8080/translate
 		? text=<текст>
 		& from=<язык с которого происходит перевод>
 		& to=<язык на который надо перевести>
	б) Воспользоваться формой-примером, которая расположена по адресу:
		localhost:8080/

3. Для того, чтобы просмотреть данные из Базы данных о визитах (создано для проверки работоспособности БД), можно воспользоваться следующим адресом:
	localhost:8080/visits
